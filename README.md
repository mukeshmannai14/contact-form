# Contact Form

A responsive Contact Form built using **HTML**, **CSS**, and **JavaScript**. This project validates user input in real time and provides clear error messages before allowing the form to be submitted.

## 🚀 Features

* Responsive design
* Full Name validation
* Phone Number validation (10 digits)
* Email validation
* Password validation

  * Minimum 8 characters
  * Uppercase letter
  * Lowercase letter
  * Number
  * Special character
* Real-time validation using JavaScript
* Success message after successful form submission
* User-friendly error messages

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)
* Font Awesome (Icons)
